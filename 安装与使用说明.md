# 雅思 7.5 冲刺 · 安装与使用说明

一个**完全离线**的安卓学习 App,8 周(56 天)每天把 **背单词 + 阅读 + 听力 + 写作 + 口语** 五项全练,目标冲刺雅思 7.5。

打包好的安装包就是项目根目录里的 **`雅思7.5冲刺.apk`**。

---

## 一、把 APK 装到手机上

### 第 1 步:把 apk 传到手机
任选一种:
- **微信/QQ 传文件**:把 `雅思7.5冲刺.apk` 发到「文件传输助手」,在手机上点开。
- **数据线**:手机连电脑,把 apk 拷到手机「下载」文件夹。
- **网盘/AirDrop 类**:上传后在手机下载。
- **adb(电脑已连手机并开USB调试)**:
  ```
  ~/Library/Android/sdk/platform-tools/adb install 雅思7.5冲刺.apk
  ```

### 第 2 步:允许安装未知来源的应用
安卓默认禁止安装非应用商店的 apk。点开 apk 时若提示「禁止安装」:
1. 进入 **设置 → 应用 → 特殊应用权限 → 安装未知应用**(不同手机叫法略有不同:小米叫「纯净模式」、华为叫「外部来源应用下载」);
2. 找到你用来打开 apk 的那个 App(如「文件管理」「微信」「浏览器」),把 **允许安装** 打开。

### 第 3 步:安装
回到 apk 点 **安装** → **打开**。桌面会出现「雅思7.5冲刺」图标。

> 如果弹出「Play 保护机制」警告,选「仍要安装 / 详情→仍然安装」即可。这是因为没在官方商店上架,属正常现象。

## 二、首次使用注意

- **第一次打开当天即为 Day 1**,之后每天进度自动 +1,共 56 天。
- **听力朗读**:用的是手机系统的语音合成(TTS)。多数手机自带「Google 文字转语音」。若点播放没声音:
  设置 → 系统 → 语言和输入法 → 文字转语音输出 → 安装/选择「Google 文字转语音」,并下载英语语音包。听力页也始终可「显示原文」。
- **口语录音**:第一次点录音会请求麦克风权限,选「允许」。
- **和 Gemini 对练**:口语页点「复制提示词并打开 Gemini」,会复制好考官提示词并跳转 Gemini App(需先装好 Gemini),粘贴发送即可开始口语模拟考。

## 三、每天怎么用(建议 70–80 分钟)

打开「今日」页,从上到下把 5 个任务做完,做完自动打卡:

| 模块 | 做什么 | 用时 |
|------|--------|------|
| 🗂️ 背单词 | 复习到期单词 + 学当天新词(间隔重复) | ~15 min |
| 📖 阅读 | 读文章做题,看解析 | ~20 min |
| 🎧 听力 | 听 TTS 朗读做题,可调语速、先别看原文 | ~15 min |
| ✍️ 写作 | 按结构和句型写,再对照范文和自查清单 | ~20 min |
| 🗣️ 口语 | 看题卡,用录音自评,或跳 Gemini 对练 | ~10 min |

「我的」页可看连续打卡、打卡日历、各项正确率,并能调每日新词量、朗读语速口音、备份/导入进度。

## 四、内容会持续更新(怎么换新版本)

当前是 **v1**,已包含第 1 周全部内容 + 完整功能。后续会继续补充更多单词、文章、听力、题目(覆盖全 8 周)。

更新方法:拿到新的 `雅思7.5冲刺.apk`,在手机上直接覆盖安装即可——**你的学习进度保存在手机本地,不会丢失**(也可在「我的」里先「导出」备份)。

> 开发者重新打包命令(在本文件夹):`bash build_apk.sh`
